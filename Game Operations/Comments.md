# 游戏运营策略 Game Operation Strategy

## 1. 滚服游戏
典型案例：微信小程序游戏

- 充值项目要多到超乎想象，意味着其数值系统、养成机制要有非常多不同的类别。
- 游戏难度曲线一定要合适，要先让玩家爽再刁难玩家。
- 福利的噱头功夫一定要做好，先从玩家总量下手，再筛出愿意付费的玩家


## 2. 内容游戏
### 2.1 燕云十六声国际服 （网易，开放世界、武侠、剧情）
联机部分内容基础是网游的底子，各种团本、社交、服装。当然也有pvp，大逃杀、工会战、单挑。

养成方面，其实就是刷，刷数值，当然理论上顺带也能保持日活量。

单机部分，也应该是游戏的最核心，就是剧情，包括主线、侠迹、奇遇。（我一开始玩就是想看剧情）剧情设计，按我的理解，其实一直是按历史背景改编的武侠故事，当然也有一些宣传中华传统文化的部分，以及卖服装（最主要的收入来源）

依照我的经验，剧情内容可能不是主要日活玩家看重的点，这可以解释为什么燕云会有如此庞大的多人游玩项目。

主线剧情作为最为核心的内容，它的更新频率其实真的很难说（我是偏剧情党，所以我是喜欢更的快的），而且为了游戏存续，故事是很难真的写到完结。那么这些“赖着不结束”的故事内容如何写好，我不知道。（即质量、更新频率、是否完结的平衡）
